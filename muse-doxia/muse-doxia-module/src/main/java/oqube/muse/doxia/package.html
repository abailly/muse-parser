This package provides temporary support for using Muse format in Maven site documentation generation process based on Doxia.

To use, simply put documentation in muse format in a directory named muse where maven will find it (usually src/site/ if your project follows standard maven layout convention.

Each file should be suffixed .muse and should of course follows the formatting conventions of muse.
